# GeoVision-LULC 🌍

A state-of-the-art deep learning system for Land Use Land Cover (LULC) semantic segmentation using Sentinel-2 satellite imagery.

## 🚀 Research Highlights
This project demonstrates the transition from traditional CNN-based segmentation to modern **Vision Transformers (ViT)** for geospatial analysis. Our findings show that Transformer architectures (SegFormer) achieve higher accuracy while being significantly more computationally efficient for satellite imagery tasks.

### 📊 Performance Benchmark
| Model | mIoU | Size (MB) | Parameters | Advantage |
|-------|------|-----------|------------|-----------|
| **SegFormer (b0)** | **0.4614** | **44.8 MB** | ~3.7 M | **Winner**: SOTA performance, ultra-lightweight |
| **DeepLabV3+** | 0.4243 | 476.1 MB | ~59 M | Strong spatial features, but heavy |

### 🔍 Explainability
Integrated **GradCAM** support allows for visual auditing of model focus. We visualize exactly which spatial features (e.g., riverbanks, urban density) trigger specific class activations, ensuring a transparent and trustable AI pipeline.

## 🛠️ Features
*   **Modular Architectures:** Seamless switching between UNet, DeepLabV3+, SegFormer, and more.
*   **Geospatial Intelligence:** Native `rasterio` support for GeoTIFF export and coordinate preservation.
*   **Production Dashboard:** Interactive Streamlit app for real-time inference and overlay review.
*   **Research Arsenal:** Automated generation of LaTeX tables, Confusion Matrices, and Qualitative grids.

## 📦 Installation
Managed with `uv` for reproducible environments.
```bash
git clone https://github.com/username/geovision-lulc.git
cd geovision-lulc
uv sync
source .venv/bin/activate
```

## 🚀 Usage

### 1. Training
Launch highly-configurable training runs via YAML configs:
```bash
python scripts/train.py --config configs/segformer.yaml
```

### 2. Export Research Artifacts
Generate the final figures for your paper or portfolio:
```bash
python scripts/export_research_artifacts.py
```
This produces:
*   `results/qualitative_showcase.png` (Side-by-side segmentation results)
*   `results/confusion_matrix.png` (Class-wise performance audit)
*   `results/efficiency_comparison.png` (mIoU vs. Size chart)

### 3. Streamlit Dashboard
```bash
streamlit run app/streamlit_app.py
```
